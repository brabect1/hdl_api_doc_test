4 Overview
5 --------
6 
7 This UVC (UVM Verification Component) implements a simple APB bus agent.
8 The agent consists of lower level components that yield initiating and
9 monitoring bus transactions. The agent's structure is captured in the
10 figure below.
11 
12 Components of the agent:
13 
14 - Driver (class apb_driver)
15 - Monitor (class apb_monitor)
16 - Sequencer (class apb_sequencer)
17 - Coverage monitor (class apb_coverage_monitor)
18 
19 The agent may operate in an active mode, in which it instantiates all the
20 usual UVC comonents (driver, sequencer, monitor). In a passive mode, the
21 agent instantiates only the monitor. The operation mode as well as other
22 features are controlled through an agent's configuration object (class
23 apb_agent_config).
24 
25 The agent may be paired with a UVM register model through an adapter class
26 reg2apb_adapter.

31 Basic Use
32 ---------
33 
34 The following code shows the basic integration of the APB agent into an environment.
35 
36 ```
37 class my_env extends uvm_env;
38 
39 apb_agent_pkg::apb_agent m_apb_agent;
40 
41 function void build_phase(uvm_phase phase);
42  apb_agent_pkg::apb_agent_config l_apb_agent_cfg;
43 
44  // setup agent's configuration
45  l_apb_agent_cfg = ...
46 
47  // instantiate the agent
48  uvm_config_db #(apb_agent_pkg::apb_agent_config)::set(this, "m_apb_agent*", "apb_agent_config", l_apb_agent_cfg);
49  m_apb_agent = apb_agent::type_id::create("m_apb_agent", this);
50 
51  ...
52 endfunction: build_phase
53 
54 function void connect_phase(uvm_phase phase);
55  ...
56  // Set up the agent sequencers as resources:
57  uvm_config_db #(apb_agent_pkg::apb_sequencer)::set(null, "*", "apb_sequencer", m_apb_agent.m_sequencer);
58  ...
59 endfunction: connect_phase
60 
61 endclass: my_env
62 ```

65 Using Register Model
66 --------------------
67 
68 Some extra code is needed to pair the agent with a register model.
69 
70 ```
71 class my_env extends uvm_env;
72 
73 my_env_config m_cfg; // environment config (includes a register model `my_rm`)
74 
75 uvm_reg_predictor#(apb_agent_pkg::apb_seq_item) apb2reg_predictor; // we will use an explicit predictor
76 
77 function void spi_env::build_phase(uvm_phase phase);
78  if(!uvm_config_db #(my_env_config)::get(this, "", "my_env_config", m_cfg)) begin
79  `uvm_error("build_phase", "Failed to find my_env_config")
80  end
81  ...
82  // build apb_agent
83  ...
84  // Build the register model predictor
85  apb2reg_predictor = uvm_reg_predictor #(apb_agent_pkg::apb_seq_item)::type_id::create("apb2reg_predictor", this);
86  ...
87 endfunction:build_phase
88 
89 function void connect_phase(uvm_phase phase);
90  if(m_apb_agent.m_cfg.active == UVM_ACTIVE) begin
91  apb_agent_pkg::reg2apb_adapter reg2apb;
92  reg2apb = reg2apb_adapter::type_id::create("reg2apb");
93 
94  // Only set up register sequencer layering if the top level env
95  if(m_cfg.my_rm.get_parent() == null) begin
96  m_cfg.my_rm.APB_map.set_sequencer(m_apb_agent.m_sequencer, reg2apb);
97  end
98 
99  // Replacing implicit register model prediction with explicit prediction
100  // based on APB bus activity observed by the APB agent monitor
101  // Set the predictor map:
102  apb2reg_predictor.map = m_cfg.my_rm.APB_map;
103  // Set the predictor adapter:
104  apb2reg_predictor.adapter = reg2apb;
105  // Disable the register models auto-prediction
106  m_cfg.my_rm.APB_map.set_auto_predict(0);
107  // Connect the predictor to the bus agent monitor analysis port
108  m_apb_agent.ap.connect(apb2reg_predictor.bus_in);
109  end
110  ...
111 endfunction: connect_phase
112 
113 endclass: my_env
